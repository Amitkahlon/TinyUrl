# TinyUrl

Automation example for a simple TinyUrl website. 

## Project Architecture
Tests: 
- E2E Tests: Cypress.
- HTTP Tests: Mocha.
- Unit Tests: Jest.

Application:
- Server: Node express.
- UI Framework: React.
- Project Management: Monorepo using Lerna.
